- Nginx前端部署

Nginx目录：
```
启动
cd  /usr/local/nginx/sbin/
./nginx

前端文件存放目录
/usr/local/nginx/web/

login 框架前端
pigon 参数业务前端
```
`前端更新只需将原有dist删除再把打好的dist包放到相应目录下即login或pigon目录下`
